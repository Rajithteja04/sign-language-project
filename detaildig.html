System Architecture

MediaPipe Holistic Pipeline

Landmark extraction & feature vector generation for gesture recognition

🎥
01 · Input Live Webcam Video Stream

Continuous real-time video captured from the user's webcam. Each frame is passed downstream for processing.

Continuous frames Real-time capture RGB video
🖼️
02 · Capture Frame Capture Layer

Webcam captures live frames at regular intervals. Each frame is decoded and buffered for landmark detection.

Frame decode Buffer management Preprocessing
🧠
03 · Detection MediaPipe Holistic Landmark Model

Simultaneously detects keypoints across three anatomical regions using ML-based holistic pose estimation.

33
Pose

Full-body joint landmarks including torso, limbs & hips

468
Face

Dense facial mesh keypoints for expression tracking

21×2
Hands

Finger joint landmarks per hand for gesture detail

🦴
04 · Visualization Skeletal Landmark Overlay

Detected landmarks are rendered as an annotated skeleton over the live video frame for real-time visual feedback and debugging.

Body skeleton Hand wireframe Face mesh Real-time overlay
📐
05 · Normalization Coordinate Normalization Stage

Raw landmark coordinates are extracted, scaled, and normalized to produce consistent, scale-invariant spatial representations across different signers and camera setups.

x, y, z extraction Scale invariance Min-max / unit norm Visibility filtering
🧮
06 · Encoding Feature Vector Generation

All normalized landmark coordinates are flattened and concatenated into a single structured numerical vector capturing the full spatial configuration of the signer per frame.
